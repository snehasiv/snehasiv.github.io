  • SLEEPWELL: Maximize power savings in Tree-like Computer Networks

    Algorithms, Computer Networks May'15 - Nov'15

    Algorithm developed focused on tree structured computer networks used in developing countries in India. Divides user in profiles based on network usage time slots till access layer, automated to go to ‘sleep mode’ whenever each user is offline.

    Implementation benefit analysis done by 50day+ data collected over IITK network. Pressure points included developing heuristics for developing and proving real life ideal implementation of the Sleepwell network.

  • A Mask based approach for Lossless Colour Image Steganography

    Image Processing- Steganography Published in Aug'15

    Published in AMCIS -2015 (America’s Conference on Information Systems). Lossless recovery with high payload capacity and low error magnitude measured through PSNR values.

    Provides multi-level security using constant key with XOR-ing LSB pixel bits in masks of 4x4 and 9x9 with Arnold Transform. Code Implemented on MATLAB.
